# mazegen-forge

Standalone Python library for generating and solving mazes.  
No UI, no external dependencies — pure Python 3.11+.

## Algorithms

| Role      | Options                     |
|-----------|-----------------------------|
| Generator | `"DFS"`, `"PRIM"`, `"KRUSKAL"` |
| Solver    | `"BFS"` (shortest path), `"DFS"` |

## Installation

```bash
pip install mazegen_forge-1.0.0-py3-none-any.whl
```

> The exact filename generated by `python -m build` will be `mazegen_forge-1.0.0-py3-none-any.whl`.

## Quick Start

```python
import mazegen_forge as mg

# Generate a 20×15 maze with a fixed seed using the Prim algorithm
maze = mg.generate(20, 15, seed=42, algorithm="PRIM")

# Solve it (BFS → guaranteed shortest path)
path = mg.solve(maze, algorithm="BFS")

print(f"Solution length: {len(path)} steps")
print(f"First cell: {path[0]}, last cell: {path[-1]}")
```

## API Reference

### `generate_maze(...) → Maze`

| Parameter   | Type               | Default          | Description                              |
|-------------|--------------------|------------------|------------------------------------------|
| `width`     | `int`              | —                | Number of cells horizontally             |
| `height`    | `int`              | —                | Number of cells vertically               |
| `entry`     | `tuple[int, int]`  | `(0, 0)`         | Entry point coordinates                  |
| `exit_`     | `tuple[int, int]`  | `(w-1, h-1)`     | Exit point coordinates                   |
| `seed`      | `int \| None`      | `None` (random)  | Seed for reproducible mazes              |
| `algorithm` | `str`              | `"DFS"`          | Generator: `"DFS"`, `"PRIM"`, `"KRUSKAL"` |
| `perfect`   | `bool`             | `True`           | `False` removes some walls (loops)       |

### `solve_maze(maze, algorithm="BFS") → list[tuple[int, int]]`

Returns the solution path as a list of `(x, y)` cell coordinates from entry to
exit, or an empty list if no path exists.

### Direct class usage

```python
from mazegen_forge import Maze, KruskalGenerator, BFSSolver

maze = Maze(width=10, height=10, entrypoint=(0, 0), exitpoint=(9, 9))
KruskalGenerator(maze).generate(perfect=True)

solver = BFSSolver(maze)
solver.solve()
print(solver.path)   # list of (x, y) tuples
```

### Accessing the grid

```python
from mazegen_forge import Direction

cell = maze.grid[y][x]
print(cell.has_wall(Direction.N))   # True / False
print(cell.walls)                   # Direction bitmask (N=1, E=2, S=4, W=8)
print(cell.type)                    # CellType.NORMAL / ENTRYPOINT / EXITPOINT / TEMPLATE
```
